## 中文

适用于**1.20版本**的AE-UI材质包,包含一定文本修正,贴图修正.

为大量模组添加了AE风格的UI,分亮色和暗色2个版本,例如: **农夫乐事**,**精妙存储**,**无尽贪婪**...,其他mod的UI加紧制作中.

### 注

特别针对[GTO整合包](https://gtodyssey.com/zh-hans/)进行适配

### Q&A

- Q:如何使用本材质包</br>
  A:将材质包放在资源包界面最上面,修改过很多内容,与其他AE的UI材质包不兼容

### English

The AE-UI texture pack for **version 1.20** includes certain text and texture corrections

We have added AE-style UIs for a large number of mods, available in both light and dark versions, such as **Farmers' Delight**, **Sophisticated**, **Avaritia**, and so on. The UIs for other mods are currently in the pipeline

### Note

Specifically adapted for the [GTO(GregTech-Odyssey)](https://gtodyssey.com/zh-hans/)

### Q&A

- Q: How to use this texture pack</br>
  A: Place the resource package at the top of the resource package interface. It has undergone many modifications and is not compatible with other AE UI resource packages
